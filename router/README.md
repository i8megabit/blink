# router

LLM Router - интеллектуальная маршрутизация запросов к оптимальным моделям

## Запуск

```bash
make build
make run
```

## Тестирование

```bash
make test
```

## API

- Health check: http://localhost:8004/health
- Swagger UI: http://localhost:8004/docs
- Metrics: http://localhost:8004/metrics
